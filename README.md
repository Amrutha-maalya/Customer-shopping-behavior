# Customer  Behavior Data Analytics Portfolio Project

## Overview

This project demonstrates an end-to-end data analytics workflow, starting from raw data ingestion to business insights visualization. The process includes data loading and cleaning, SQL-based analysis, dashboard creation, and presentation of findings.

The project workflow covers:

* Loading and exploring the dataset using Python
* Data cleaning and preprocessing in Jupyter Notebook
* Data analysis using MySQL queries
* Interactive dashboard development in Power BI
* Business insights presentation using Gamma
* 

## Tools & Technologies

| Tool             | Purpose                                |
| ---------------- | -------------------------------------- |
| Python           | Data loading and preprocessing         |
| Jupyter Notebook | Data cleaning and exploratory analysis |
| MySQL            | Querying and analytical operations     |
| Power BI         | Dashboard creation and visualization   |
| Gamma            | Presentation and storytelling          |

## Project Steps

### 1. Data Loading

* Imported dataset using Python.
* Performed initial data inspection.
* Checked data types and structure.

### 2. Data Cleaning

* Handled missing values.
* Removed duplicate records.
* Corrected data inconsistencies.
* Prepared clean data for analysis.

### 3. SQL Analysis

* Imported cleaned data into MySQL.
* Executed analytical queries.
* Generated key metrics and summaries.
* Answered business-related questions using SQL.

### 4. Dashboard Development

* Connected Power BI to the processed dataset.
* Created visualizations and KPIs.
* Built an interactive dashboard for business insights.

### 5. Presentation

* Summarized findings and recommendations.
* Created a professional presentation using Gamma.

## How to Run

### Prerequisites

* Python 3.x
* Jupyter Notebook
* MySQL Server
* Power BI Desktop

### Steps

1. Clone the repository.
2. Install required Python libraries.
3. Open the Jupyter Notebook and run the data cleaning process.
4. Import the cleaned dataset into MySQL.
5. Execute the SQL queries.
6. Open the Power BI file to view the dashboard.
7. Review the Gamma presentation for project insights.
